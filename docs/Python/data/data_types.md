Python, a diferencia de otros lenguajes, son de *tipado dinámico*. Esto quiere decir que no se declara el tipo de la variable y los mismos se dividen en 4:

#### Enteros (int)
```python
a = int(1) 
#1
```
#### Reales (float)
```python
b = float(1) 
#1.0
```
#### Booleano (bool)
```python
c = bool(1) 
#True
```
Si o no, 0 o 1, True or False

#### Cadenas de texto (str)
```python
d = str(1) 
#'1'
```
